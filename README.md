# Shoez Shop (Personal Project)

A full-stack e-commerce web application for shoes, built as a personal project to demonstrate my skills in both frontend and backend development.

## Tech Stack

### Frontend
- **Vue.js** – JavaScript framework for building interactive web interfaces.
- **TailwindCSS** – Utility-first CSS framework for fast and responsive styling.

### Backend
- **FastAPI** – Python framework for building fast and maintainable REST APIs.
- **MongoDB** – NoSQL database for flexible data storage.
- **Redis** – In-memory data store for caching and session management.
- **JWT (JSON Web Token)** – Authentication and authorization.
- **WebSocket** – Real-time communication for notifications or chat.
- **FastAPI-Mail** – Sending emails from the backend.

### Version Control
- **Git** – Source code management and version control.

## Features
- User registration, login, and JWT-based authentication.
- Product management (CRUD) for shoes.
- Shopping cart and checkout system.
- Real-time notifications using WebSocket.
- Email notifications via FastAPI-Mail.
- Responsive and modern UI with Vue.js + TailwindCSS.
